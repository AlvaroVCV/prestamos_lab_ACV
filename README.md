Este es un proyecto Django diseñado como arquitectura de un sistema de inventario de notebooks. 
Tiene la posibilidad de poder cambiar el estado de los notebooks disponibles en "Nuevo", "Usado" y "en Mantención".

Consta de las caracteristicas minimas que todo libro debee tener: Código, Marca, Modelo, Estado, Disponible, Fecha de registro. 
También tienee un sistema de usuarios con la finalidad de restringir el acceso al inventario de los notebooks.
